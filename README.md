Was du bekommst:
Ein animiertes Feuerwerk-Effekt für deine Nextcloud-Instanz, der:

Raketen die hochfliegen als bunte Punkte 

Explosionen oben wo die Raketen landen 

Automatisch alle 1.5 Sekunden Raketen startet

Interaktiv ist - Click überall = Rakete von der Stelle

Trails hinter den Raketen (leichte Spur)


Schritt 1: JSLoader installieren
Als Admin in deiner Nextcloud anmelden

Apps → Suche "JSLoader" → Installieren

Admin-Einstellungen öffnen

Schritt 2: Script einfügen
Im Admin-Panel ganz nach unten scrollen

Punkt "Additional JavaScript" oder "JSLoader" suchen

Den gesamten Code aus fireworks-nc.js kopieren

Speichern klicken

FERTIG! ✨
